## Learn More
去中心化密码工具

### 支持的功能
- 密码分组
- 团队创建
- 密码团队分享
- 好友管理

### 系统流程
- 用户授权登录
- 进入系统主界面-包含创建密码, 创建密码分组, 创建团队, 授权访问, 查看密码等功能
- 其他用户通过分享链接申请加入团队, 由团队创建人审批
